GENIALY

¿QUE ES?

Genially es una plataforma web española que permite crear contenido digital interactivo y animado, como presentaciones, infografías, juegos y más, sin necesidad de conocimientos de programación. Es una herramienta versátil que ofrece una amplia variedad de plantillas y recursos para facilitar la creación de contenido atractivo y dinámico tanto para uso educativo como profesional.A diferencia de las presentaciones tradicionales, Genially permite agregar animaciones, interacciones y elementos multimedia para hacerlas más atractivas y dinámicas,facilita la creación de infografías informativas y visualmente atractivas, con la posibilidad de agregar interactividad para que el usuario pueda explorar la información de manera más profunda.La plataforma ofrece la posibilidad de crear juegos educativos, cuestionarios y otros tipos de actividades interactivas para evaluar el aprendizaje de manera divertida y dinámica. es una herramienta versátil que facilita la creación de contenido interactivo y animado para diversos fines, desde presentaciones y materiales educativos hasta infografías y juegos, todo ello sin requerir conocimientos técnicos complejos.

¿PARA QUE SIRVE?

¿COMO REALIZAR UNA IMAGEN INTERACTIVA?

1. Crea un nuevo Genially o utiliza una plantilla: Puedes comenzar con una plantilla pre-diseñada o crear un lienzo en blanco

2. Sube tu imagen Haz clic en "Crear imagen interactiva" y selecciona tu imagen desde tu equipo, URL, o servicios como Google Drive o Dropbox.

3. Añade elementos interactivos: Selecciona la herramienta de puntos de acceso(botones,inconos,etc) y colocalos estrategicamente sobre la imagen

4. Configura la interactividad: has clic en cada punto de acceso y elige la accion que deseas que realice

5.Personaliza la apariencia: Cambia el tamaño,color,forma y animacion de los puntos de acceso para que coincida con tu diseño

6. Añadir contenido multimedia: Inserta videos,audios,imagenes o texto dentro de las ventanas emergentes o etiquetas para enriqueser la experiencia

7. Publicar y compartir:Una vez que estessatisfecho con tu imagen interactiva,publicala y comparte el enlace con tu audiencia
